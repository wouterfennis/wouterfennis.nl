Smartphone Comparer

Voor iPhone.nl heb ik een vergelijker gemaakt om iPhones zonder abbonement te vergelijken

Een interessante uitdaging tijdens dit project was het combineren van de verschillende API's van de aanbieders tot één databak. Om vervolgens daaruit de vergelijker de iPhones te laten ophalen.

Om vanuit verschillende XML API's een mal te vormen voor eenduidige data heb ik zogenaamde filters gemaakt per API. Elk filter weet precies hoe de XML naar het juiste formaat omgezet moet worden. Alle gegevens worden vervolgens naar een database geschreven.

Het verwerken van de gegevens is volledig in PHP geschreven. De vergelijker is met jQuery gerealiseerd.

Deze opdracht heb ik destijds voor mijn werkgever All Automation Service B.V. gemaakt.

Roman Numeral Converter

Tijdens een gesprek met een studiegenoot over het Romeinse getallenstelsel kreeg ik het idee om app te maken die beide richtingen tussen decimaal en romeins kan rekenen.

Omdat ik de applicatie eventueel op zowel iOS als Android zou willen uitrollen had ik gekozen om het een Cordova/Phonegap applicatie te maken. De logica zou in AngularJS worden geschreven en de front-end zou met HTML5 en CSS3 gemaakt worden.

Uiteindelijk heb ik er voor gekozen om de applicatie meer richting het "Leren van het romeinse getallenstelsel" te laten gaan. Waarbij je een willekeurig berekend romeins cijfer te zien krijgt en het antwoord moet invullen. Maar ook de andere kant op zodat je bij een decimaal getal romeinse cijfers moet combineren.

Letter Dictionary

Voor een schoolopdracht heb ik een Big Data applicatie gemaakt die aan kan geven hoe groot de kans is dat een zin bijvoorbeeld Engels is.

De applicatie heeft eerst een referentie tekst nodig. Dus je laat hem eerst een tekst inlezen die 100% engels is. Bijvoorbeeld Alice in Wonderland. Vervolgens gaat de applicatie kijken op letter niveau hoe vaak bepaalde letters na elkaar komen. Het zal bijvoorbeeld niet vaak voorkomen dat de Z na een B komt.

Met de referentietabel in gedachten gaat de applicatie een willekeurige tekst lezen en geeft per regel aan hoe groot de kans is dat dit Engels is.

Je kan elk soort referentie tekst meegeven en je bent dus niet gebonden aan een taal. Deze applicatie is gemaakt met Hadoop
